Shortcut guide

  • H or SHIFT + ←
    Switch to the previous file
  • L or SHIFT + →
    Switch to the next file
  • h or ←
    Scroll the page to the left
  • l or →
    Scoll the page to the right
  • k or ↑
    Scroll the page up
  • j or ↓
    Scroll the page down
  • u or PgUp
    Scroll up by one page
  • d or PgDn
    Scroll down by one page
  • g
    Scroll to the top of the page
  • G
    Scroll to the bottom of the page
  • b
    Toggle snap to bottom on file change
  • f
    Open the filepicker
  • c
    Close the active file
  • o + Link ID
    Open/Click a link
  • q
    Exit the application
  • e
    Edit the active file
  • ?
    Open/Close (this) shortcut guide
  • s
    Toggle swicth to changing file
